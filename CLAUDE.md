# AtCoder 練習リポジトリ

## 環境
- Python 3.11
- パッケージ管理: uv
- タスクランナー: mise

## よく使うコマンド
| やること | コマンド |
|---------|---------|
| コンテスト設定 | `mise run sc` |
| 解答実行 | `mise run p` |
| 解答保存 & commit | `mise run sv` |
| フォーマット | `mise run f` |
| lint | `mise run l` |

## ディレクトリ構成
- `main.py` — 今解いている問題（作業ファイル）
- `_result/{contest}/{problem}/main.py` — 保存済み解答
- `_template/main.py` — テンプレート
